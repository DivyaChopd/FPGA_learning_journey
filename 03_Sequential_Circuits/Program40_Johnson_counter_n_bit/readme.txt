Johnson counter:

It is also called as twisted ring counter because it is same as ring counter except the feedback from last counter to first counter is not from q but from not q.

No. of states = 2 * No. of FF
For ex. if 4 FF is used then it covers 2*4 = 8 states --- Advantage over ring counter
